# Utilities View

### General
 - Utilities can be added to source as C++ or written in python
 - A bunch of left-side tabs can be used to select utility module

### Utility Modules
 - Export current instrument as standalone VST
 - Machine learning sample processing
   - De-noise
   - Isolate vocals, bass, drums, etc
 - Auto sampler for VST
 - Auto sampler for hardware
 - Auto model an amplifier
 - Auto model an effect with deep learning
 - Capture a room impulse response
