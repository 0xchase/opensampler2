# opensampler

### General
 - Has special package manager

### Layout
 - Main view is the *instrument view*, which can be flipped to the *modular view*
 - Button on top to open *browser view*
 - Another window for the *utilities view*

---

### Browser View
 - Browse different instruments according to their tags in various folders
 - Can view instrument description with links, preview image, preview audio

### Instrument View
 - Automatically generated from modular view
 - Can select presets from the top here

### Modular View
 - Animate wires to show when signal is passing through
 - Can select presets from the top here

### Utilities View
 - Export current instrument as standalone VST
 - Machine learning sample processing
 - De-noise
 - Isolate vocals, bass, drums, etc
 - Auto sampler for VST
 - Auto sampler for hardware
 - Auto model an amplifier
 - Auto model an effect with deep learning
 - Capture a room impulse response
